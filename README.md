# MyQQ:一个基于egg.js和react全家桶的即时通讯移动端网页

## 项目概述：

一个具备聊天功能和空间的即时通讯软件。

## 需求分析

1. 注册登录 √
2. 个人信息的完善和展示 √
3. 好友模块：好友的添加 √
4. 好友个人聊天功能 √
5. 群聊功能 
6. 空间查看功能
7. 空间点赞功能
8. 空间动态发布

## 设计参考

参考QQ：简洁模式

## 后端框架

egg.js

## 前端技术栈

React全家桶
